### 문제 설명

<p>서현이는 밤(night)이 싫고 낮이 좋지만, KCPC 검수진들이 밤에 주로 활동해 서현이는 화가 났다.</p>

<p>서현이는 본인의 체스판에 있던 나이트의 이름이 마음에 들지 않는다는 이유로 나이트의 움직임을 막고 싶어졌다. 나이트는 한 번 이동하여 가로로 $2$칸, 세로로 $1$칸 떨어진 곳이나 가로로 $1$칸, 세로로 $2$칸 떨어진 곳으로 한 번에 이동할 수 있다. 서현이는 장애물을 배치하여 나이트의 움직임을 막으려 한다. 장애물을 배치한 칸에는 나이트가 이동할 수 없다.</p>

<p>구체적으로 현재 나이트는 무한한 체스판의 $\left( 0,0 \right)$에 위치하고 있으며, 최종적으로 $\left( x,y \right)$로 움직이려 한다. 서현이가 장애물을 두기 전 나이트가 $\left( x,y \right)$에 도착하기 위해 이동해야 하는 최소 횟수보다, 몇 개의 장애물을 둔 후 이동해야 하는 최소 횟수를 더 많게 만들고 싶다. 단, $\left( 0,0 \right)$과 $\left( x,y \right)$에는 장애물을 둘 수 없으며, $0$번 이상의 유한한 이동으로 $\left( x,y \right)$에 도달할 수 없게 만든다면 최소 횟수를 $10^{20\, 070\, 817}$회로 생각한다. 장애물을 두기 전에는 체스판의 모든 칸에 항상 $0$번 이상의 유한한 이동으로 도착할 수 있다.</p>

<p>체스판 위의 도착점 $\left( x,y \right)$가 주어질 때, 규칙을 만족하기 위해 서현이가 배치해야 하는 장애물의 최소 개수를 구해보자.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T(1\le T\le 100\, 000)$이 주어진다.</p>

<p>두 번째 줄부터 $T$ 줄에 걸쳐 각 테스트 케이스별로 도착점을 의미하는 두 정수 $x,y(-10^{18}\le x,y\le 10^{18})$가 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄부터 $T$ 줄에 걸쳐 각 테스트 케이스별로 서현이가 설치해야 하는 장애물의 최소 개수를 출력한다. 단, 조건을 만족하게 장애물을 설치할 수 없다면 대신 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>
